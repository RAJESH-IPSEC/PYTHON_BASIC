# 🔍 Check Python Version

You can verify that Python is installed on your system by running the following command in the terminal or command prompt.

```bash
python --version
```

Example (Windows):

```bash
C:\Users\YourName> python --version
Python 3.x.x
```

---

# 💻 Python Command Line

For quick testing, Python can run directly in the command line without creating a `.py` file.

Start the Python interactive shell:

```bash
python
```

Example:

```bash
C:\Users\YourName> python
```

You will see something similar to:

```text
Python 3.x.x (default, ...)
>>>
```

Now you can run Python code directly.

Example:

```python
print("Hello, Python!")
```

---

# 🧾 Check Python Version Using Code

You can also check the Python version from inside a Python script using the **sys** module.

```python
import sys

print(sys.version)
```

Output example:

```text
3.x.x (main, ...)
```

---

# 🚪 Exit the Python Command Line

When you are done using the Python interactive shell, exit using:

```python
exit()
```

or press:

```text
Ctrl + Z  (Windows)
Ctrl + D  (Linux/macOS)
```

---

✅ **Tip:** The Python command line is useful for quickly testing small pieces of code without creating a full script file.
